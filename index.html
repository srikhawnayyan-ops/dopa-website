กรมการปกครอง

Department of Provincial Administration

เอกสารฉบับนี้เป็นกฎระเบียบของกรมการปกครองที่กำหนดระเบียบ วินัย ข้อบังคับ อัตราโทษ และบทลงโทษ สำหรับข้าราชการและเจ้าหน้าที่ทุกนายในสังกัดกรมการปกครอง โดยให้ถือปฏิบัติโดยเคร่งครัด และให้ใช้บังคับร่วมกับกฎหมายหรือพระราชบัญญัติที่เกี่ยวข้อง ทั้งนี้ ผู้ใดฝ่าฝืนจะถูกพิจารณาลงโทษ ตามระเบียบหรือกฎหมายที่เกี่ยวข้องตามความเหมาะสม

หมวดที่ ๑ บททั่วไป

ข้อ ๑ ระเบียบนี้เรียกว่า “ระเบียบกรมการปกครอง ว่าด้วยวินัย การกระทำผิด การสอบสวน และการลงโทษ”

ข้อ ๒ ระเบียบนี้บังคับใช้แก่ข้าราชการ เจ้าพนักงาน และเจ้าหน้าที่ทุกประเภทในสังกัดกรมการปกครอง

ข้อ ๓ ข้าราชการและเจ้าหน้าที่ต้องรักษาวินัยโดยเคร่งครัด ปฏิบัติตามคำสั่งของผู้บังคับบัญชา และปฏิบัติหน้าที่ด้วยความซื่อสัตย์สุจริต เที่ยงธรรม

ข้อ ๔ การกระทำใดๆ ที่เป็นการฝ่าฝืนหรือไม่ปฏิบัติตามระเบียบนี้ ให้ถือว่าเป็นการกระทำผิดวินัย

ข้อ ๕ รองอธิบดี และคณะบริหารต้องปฏิบัติราชการที่อยู่ใต้อำนาจบังคับบัญชาของอธิบดีกรมการปกครอง และต้องปฏิบัติราชการด้วยความสุจริต หากมีการกระทำที่ฝืนจริยธรรมของคณะบริหาร จะต้องธำรงวินัย ตามฐานความผิดที่ได้กระทำไว้ และต้องยึดถือการวินิจฉัยของอธิบดีเป็นหลัก และห้ามกระทำการใดที่สื่อถึงการหลบหนีโทษและความผิดทางวินัย หากมีการฝ่าฝืนอย่างเด็ดขาดจะส่งผลให้พ้นจากตำแหน่ง และรายงานพฤติกรรม

ข้อ ๖ การลงโทษข้าราชการให้เป็นไปตามความร้ายแรงทุกกรณี

  • ความร้ายแรงของการกระทำความผิด
  • มูลค่าความเสียหายที่เกิดขึ้นต่อทางราชการ
  • ประวัติการกระทำความผิด
  • พฤติการณ์และเจตนาของผู้กระทำผิด
  • ดุลยพินิจของผู้บังคับบัญชา

หมวดที่ ๒ วินัยข้าราชการกรมการปกครอง

ข้อ ๗ ปฏิบัติราชการด้วยความซื่อสัตย์ สุจริต และเที่ยงธรรม

ข้อ ๘ ปฏิบัติตามคำสั่งของผู้บังคับบัญชาซึ่งสั่งในหน้าที่ราชการโดยชอบด้วยกฎหมาย

ข้อ ๙ รักษาความลับของทางราชการและหน่วยงานอย่างเคร่งครัด

ข้อ ๑๐ แต่งกายเครื่องแบบหรือชุดสุภาพตามระเบียบที่กรมการปกครองกำหนด

ข้อ ๑๑ ต้องปฏิบัติราชการเวรและออกเวรตามเวลาที่กำหนด

ข้อ ๑๒ ใช้วาจาสุภาพ ให้เกียรติเพื่อนร่วมงานและประชาชน

ข้อ ๑๓ ต้องให้ความเคารพผู้บังคับบัญชา คณะบริหาร และผู้อวุโส

ข้อ ๑๔ ห้ามใช้อำนาจหน้าที่ในทางมิชอบ

ข้อ ๑๕ ต้องไม่กระทำการใดๆ ให้หน่วยงานเสื่อมเสียชื่อเสียง

ข้อ ๑๖ ต้องไม่เปิดเผยข้อมูลภายในหน่วยงาน

ข้อ ๑๗ ห้ามปลอมแปลงเอกสาร

ข้อ ๑๘ ต้องไม่แอบอ้างตำแหน่งหรือบุคคลอื่น

ข้อ ๑๙ ต้องปฏิบัติหน้าที่ด้วยความซื่อสัตย์สุจริต และไม่แสวงหาประโยชน์ที่มิควรได้โดยชอบด้วยกฎหมาย

ข้อ ๒๐ ต้องไม่ให้การเท็จต่อผู้บังคับบัญชา

ข้อ ๒๑ ต้องไม่ใส่ความหรือกลั่นแกล้งผู้อื่น

ข้อ ๒๒ ต้องไม่ละเมิดความเป็นส่วนตัวของผู้อื่น

ข้อ ๒๓ ต้องไม่ละทิ้งหน้าที่หรือเพิกเฉยต่อหน้าที่

ข้อ ๒๔ ต้องดูแลรักษาทรัพย์สินของทางราชการ

หมวดที่ ๓ ระดับโทษทางวินัย

ข้อ ๒๕ โทษทางวินัยมีดังต่อไปนี้

  • ว่ากล่าวตักเตือนด้วยวาจา
  • ตักเตือนขั้นที่ ๑
  • ตักเตือนขั้นที่ ๒
  • ภาคทัณฑ์
  • พักราชการ
  • ลดตำแหน่ง
  • ปลดออก
  • บัญชีดำ

ข้อ ๒๖ การลงโทษให้พิจารณาตามความร้ายแรงของการกระทำ และผู้มีอำนาจสามารถเลือกลงโทษตามระดับที่เห็นสมควร

ข้อ ๒๗ ผู้กระทำผิดซ้ำ อาจเพิ่มระดับโทษ

ข้อ ๒๘ ความผิดร้ายแรงสามารถปลด ปลดออก หรือบัญชีดำได้ทันที

หมวดที่ ๔ ฐานความผิดทางวินัย

ส่วนที่ ๑ ความผิดเกี่ยวกับการปฏิบัติหน้าที่

ข้อ ๒๙ ข้าราชการผู้ใดละทิ้งหน้าที่ในเวลาปฏิบัติราชการ

โทษ: ว่ากล่าวตักเตือน / ตักเตือนขั้นที่ ๑ / ตักเตือนขั้นที่ ๒

ข้อ ๓๐ ไม่เข้าเวร หรือออกจากเวรก่อนเวลา

โทษ: ตักเตือนขั้นที่ ๑ / ตักเตือนขั้นที่ ๒ / ภาคทัณฑ์ / พักงาน

ข้อ ๓๑ ขาดเวรโดยไม่แจ้งลาล่วงหน้า

โทษ: ตักเตือนขั้นที่ ๒ / ภาคทัณฑ์ / พักงาน

ข้อ ๓๒ เพิกเฉยต่อหน้าที่

โทษ: ตักเตือนขั้นที่ ๒ / ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง

ข้อ ๓๓ ละทิ้งหน้าที่

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด

ส่วนที่ ๒ ความผิดเกี่ยวกับความประพฤติ

ข้อ ๓๔ แต่งกายไม่เรียบร้อย หรือไม่ถูกต้องตามระเบียบ

โทษ: ว่ากล่าวตักเตือน / ตักเตือนขั้นที่ ๑ / ตักเตือนขั้นที่ ๒

ข้อ ๓๕ ใช้คำพูดไม่สุภาพ ด่าทอ หรือสร้างความเสียหายแก่หน่วยงาน

โทษ: ตักเตือนขั้นที่ ๒ / ภาคทัณฑ์ / พักงาน

ข้อ ๓๖ ไม่เคารพผู้บังคับบัญชา

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง

ข้อ ๓๗ ทะเลาะวิวาทภายในหน่วยงาน

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด

ข้อ ๓๘ ทำให้หน่วยงานเสียชื่อเสียง

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด / ไล่ออก

ส่วนที่ ๓ ความผิดเกี่ยวกับเอกสารและข้อมูล

ข้อ ๓๙ เปิดเผยข้อมูลภายในหน่วยงาน

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด

ข้อ ๔๐ ทำเอกสารราชการสูญหายโดยประมาท

โทษ: ตักเตือนขั้นที่ ๒ / ภาคทัณฑ์

ข้อ ๔๑ ปลอมแปลงเอกสาร หรือบันทึกข้อมูลเท็จ

โทษ: พักงาน / ลดตำแหน่ง / ปลด / ไล่ออก

ส่วนที่ ๔ ความผิดร้ายแรง

ข้อ ๔๒ แอบอ้างเป็นบุคคลอื่น

โทษ: พักงาน / ลดตำแหน่ง / ปลด / ไล่ออก / บัญชีดำ

ข้อ ๔๓ ทุจริต หรือ รับสินบน

โทษ: ปลด / ไล่ออก / บัญชีดำ

ข้อ ๔๔ ใช้อำนาจในทางมิชอบ

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด

ข้อ ๔๕ ให้การเท็จต่อผู้บังคับบัญชา หรือคณะกรรมการสอบสวน

โทษ: ภาคทัณฑ์ / พักงาน / ลดตำแหน่ง / ปลด

ข้อ ๔๖ ใส่ความ กลั่นแกล้ง หรือแจ้งข้อมูลเท็จ

โทษ: พักงาน / ลดตำแหน่ง / ปลด

ข้อ ๔๗ กระทำความผิดร้ายแรงที่ทำให้หน่วยงานเสียหายอย่างมาก

โทษ: ปลด / บัญชีดำ

หมวดที่ ๕ การสอบสวนทางวินัย

ข้อ ๔๘ มื่อมีการร้องเรียนหรือพบการกระทำผิด ให้แต่งตั้งคณะกรรมการสอบสวน

ข้อ ๔๙ คณะกรรมการสอบสวนไม่น้อยกว่า ๓ คน

ข้อ ๕๐ ผู้ถูกกล่าวหามีสิทธิ์ชี้แจง แสดงพยานหลักฐาน และให้การแก้ข้อกล่าวหา

ข้อ ๕๑ สรุปผลเสนอผู้บังคับบัญชา

ข้อ ๕๒ ผู้มีอำนาจสั่งลงโทษ คือ ผู้บังคับบัญชา หรือคณะบริหารกรม

หมวดที่ ๖ การอุทธรณ์

ข้อ ๕๓ ผู้ถูกลงโทษมีสิทธิ์ยื่นอุทธรณ์ภายใน ๓ วัน นับแต่วันที่รับทราบคำสั่ง

ข้อ ๕๔ ตั้งคณะกรรมการอุทธรณ์

ข้อ ๕๕ ผลอุทธรณ์ถือเป็นที่สุด

ข้อ ๕๖ ข้าราชการพ้นจากตำแหน่งเมื่อ

  • ลาออก
  • ถูกปลด
  • ถูกขึ้นบัญชีดำ

หมวดที่ ๘ บทเฉพาะกาล

ข้อ ๕๗ ให้ถือว่าอยู่ภายใต้ระเบียบนี้

ข้อ ๕๘ ให้ผู้บังคับบัญชาตามลำดับชั้นมีหน้าที่รักษาการตามระเบียบนี้

ข้อ ๕๙ ในกรณีที่ระเบียบนี้ไม่ได้กำหนดรายละเอียดไว้ ให้ผู้บังคับบัญชามีอำนาจพิจารณาสั่งการตามที่เห็นสมควร
